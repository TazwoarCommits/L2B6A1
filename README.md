## টাইপস্ক্রিপ্ট এ টাইপ এবং ইন্টারফেস এর মধ্যে কিছু পার্থক্য

একজন নতুন টাইপস্ক্রিপ্ট শেখা ডেভেলপার এর কাছে প্রথমাবস্থায় টাইপস্ক্রিপ্ট কিছুটা কনফিউজিং মনে হতে পারে । অনেক ধরনের ডাটা টাইপ , কখন কোন ডাটা টাইপ ব্যবহার হবে , কিভাবে টাইপ ডিক্লেয়ার করলে পরবর্তীতে সেটি নিয়ে কাজ করতে সুবিধা হবে ইত্যাদি । এর মধ্যে ডাটা টাইপ ডিক্লেয়ারেশন করার বিভিন্ন ধরন ও গঠন । তার মধ্যে দুটি হল "টাইপ" (type) এবং "ইন্টারফেস" (interface) । দুটির ই গঠন প্রায়ই এক রকম , আজ এই দুটি সম্পর্কেই বিস্তারিত জানব ।

সহজ ভাষায় বললে,

### **_ইন্টারফেস_**

* ইন্টারফেস - অবজেক্ট শেপ এবং ক্লাস এর জন্য বেশি উপযোগী ।

     interface User {  
        name: string;   
        age: number;    
      }

* ইন্টারফেস মার্জ করা যায় রিডিক্লিয়ার এর মাধ্যমে, কিন্তু টাইপ এ  রিডিক্লিয়ার সম্ভব নয় ।

       interface User {  
         name: string;  
       }
       interface User {  
         age: number;  
       }


* "extend" এর মাধ্যমে ইন্টারফেস বর্ধিত করা যায় 

       interface Animal {  
       name: string;  
       }  
       interface Dog extends Animal {  
       bark(): void;  
       }  

* ইন্টারফেস অবজেক্ট, ক্লাস এর ডাটা টাইপ এর জন বেশি উপযোগী

### **_টাইপ_**

* টাইপ - যেকোনো কিছুর জন্য ব্যবহার করা যেতে পারে। অবজেক্ট, টুপল এবং অন্য প্রিমিটিভ ডাটা টাইপ এ ।  

      type User = {
          name: string;
          age: number;  
      };  

* টাইপ ইউনিয়ন ও ইন্টারসেকশন সাপোর্ট করে ।

       type Status = "pending" | "success"   
       type Animal = { name: string };
       type Dog = Animal & { bark(): void }; 

* টাইপ ইন্টারফেস এর তুলনায় বেশি ফ্লেক্সিবল  ।  


টাইপস্ক্রিপ্ট এ টাইপ এবং ইন্টারফেস দুটির ই ক্ষেত্র বিশেষে প্রয়োজন রয়েছে। ইন্টারফেস যেখানে অবজেক্ট বা ক্লাস এর মত গঠনমূলক ক্ষেত্রে বেশি প্রয়োজনীয়, টাইপ ও তেমনি কন্ট্রোলড ও কমপ্লেক্স ডাটাটাইপ নিয়ে কাজ করতে বেশি উপযোগী ।  


## ইউনিয়ন ও ইন্টারসেকশন এর উদাহরণ  
  
  টাইপস্ক্রিপ্ট এর একাধিক ডাটাটাইপ একসঙ্গে স্ট্রিক্টলি মনিটর করার জন্য ও একাধিক শর্ত প্রয়োগের জন্য ইউনিয়ন ও ইন্টারসেকশন অত্যন্ত পাওয়ারফুল। ঠিকমত এ দুটি উপাদান ব্যবহার করতে না পারলে কোড অনেক মেসি হয়ে যায়। সেক্ষেত্রে আগে আমাদের এদের সম্পর্কে হালকা করে জেনে নেওয়া উচিত ।


  ### ইউনিয়ন( | )
  
  সহজ ভাষায়, ইউনিয়ন হলো "হয় এটা অথবা ওটা"  
  ধরা যাক, কোন একটি variable এর ডাটাটাইপ হয় স্ট্রিং অথবা নাম্বার হতে পারে ।  
     
            type A = string | number;    


    উদাহরণ :  
            function printId(id: string | number) {   
            console.log("ID:", id);
            }        
    
            printId("abc123");
            printId(987);
    
  ইউনিয়ন আমাদের সম্ভাব্য একাধিক ডাটা টাইপ হ্যান্ডেল এর সুযোগ দেয় ।     


  ### ইন্টারসেকশন( & )

  সহজ ভাষায়, ইন্টারসেকশন আমাদের একই সময় একাধিক শর্ত বাস্তবায়নে বাধ্য করে ।     
     
            type A = { name: string }    
            type B = { age: number }    
            type Person = A & B;   

    উদাহরণ :  
            type HasEmail = { email: string };   
            type HasPhone = { phone: string };   

            type ContactInfo = HasEmail & HasPhone;  
               
            const user: ContactInfo = {  
                email: "taz@example.com",  
                phone: "0123456789",  
            };
